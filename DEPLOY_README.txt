DEPLOYMENT INSTRUCTIONS:

1. Go to your Hostinger File Manager (public_html/wp-content/themes/).

2. Upload the 'vastu-2026-glass.zip' file generated below.

3. Right-click and Extract the zip file.

4. Login to WordPress Admin > Appearance > Themes.

5. You will see "Vastu 2026 Glass" - Click Activate.

6. CRITICAL: Go to Settings > Permalinks and click 'Save Changes' to flush the cache.
