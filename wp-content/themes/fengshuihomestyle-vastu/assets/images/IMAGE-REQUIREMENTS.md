# Image Requirements for Digital Zen Homepage

## Overview
This document specifies the high-resolution, minimalist lifestyle images required for the Feng Shui Homestyle Vastu website. All images should follow the "Quiet Luxury" 2025 wellness aesthetic.

## Image Standards

### General Requirements
- **Resolution:** Minimum 1920x1080px for hero images, 800x600px for cards
- **Format:** JPG or WebP (optimized for web)
- **File Size:** < 500KB each (optimized)
- **Style:** Natural morning light, serene, minimalist, high-end modern architecture
- **Color Tone:** Warm, natural, biophilic (matching our color palette)
- **Avoid:** Generic clipart, spiritual icons, overly staged photos

### Color Harmony
Images should complement our biophilic color palette:
- Warm Sand (#F5EAE1)
- Sage Green (#648E7B)
- Deep Indigo (#2E2B59)
- Earth Brown (#8B7355)

## Optimization Instructions

### Before Upload
1. **Resize** images to required dimensions
2. **Optimize** using tools like:
   - TinyPNG (tinypng.com)
   - Squoosh (squoosh.app)
   - ImageOptim (Mac)
3. **Convert** to WebP format for better performance (optional)
4. **Rename** files to match the exact names listed above

### WordPress Upload
1. Upload to: `/wp-content/themes/fengshuihomestyle-vastu/assets/images/`
2. Also upload through WordPress Media Library for easier management
3. Add descriptive Alt text for SEO and accessibility

## Fallback Behavior

Currently, the theme includes:
- SVG gradient backgrounds as fallbacks
- Graceful degradation if images are not loaded
- Proper alt text for accessibility

The site will function with the current gradient fallbacks, but real images will significantly enhance the user experience and conversion rates.

## Testing Checklist

After adding images:
- [ ] Images load on all devices (desktop, tablet, mobile)
- [ ] Loading performance is acceptable (LCP < 2.5 seconds)
- [ ] Images display correctly in all major browsers
- [ ] Alt text is descriptive and meaningful
- [ ] Images complement the color palette
- [ ] Hover effects work smoothly on cards
- [ ] Images are optimized for web (<500KB each)
