## 5. "Thumb Zone" WhatsApp Sticky Button CSS

This CSS creates a mobile-friendly, always-visible WhatsApp CTA button in the bottom-right corner (the "thumb zone" for right-handed users on mobile).

/* ==========================================================
   Thumb Zone - Sticky WhatsApp CTA Button
   For Astra Theme Footer or Custom CSS
   ========================================================== */

.whatsapp-sticky-btn {
  position: fixed;
  bottom: 24px;
  right: 24px;
  z-index: 9999;
  display: flex;
  align-items: center;
  justify-content: center;
  width: 60px;
  height: 60px;
  background: linear-gradient(135deg, #25D366 0%, #128C7E 100%);
  border-radius: 50%;
  box-shadow: 0 4px 20px rgba(37, 211, 102, 0.4);
  text-decoration: none;
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.whatsapp-sticky-btn:hover {
  transform: scale(1.1);
  box-shadow: 0 6px 28px rgba(37, 211, 102, 0.55);
}

.whatsapp-sticky-btn svg {
  width: 32px;
  height: 32px;
  fill: #ffffff;
}

  content: '';

  width: 100%;

  }

  }
}

  }
}
